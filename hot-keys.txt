Hot keys on Windows

Win + D = рабочий стол
Win + E = открыть проводник
Ctrl + T = новая вкладка в браузере
Ctrl + W = для закрытия текущего вкладку либо приложения
Alt + Tab = переход между приложениями
Ctrl + Tab = переход между вкладками браузера
Ctrl + Shift + Tab = переход обратную сторону вкладками браузера
Ctrl + N = новая вкладка в браузере(в новом приложение)
Ctrl + Shift + N = новая вкладка в браузере в режиме инкогнито(в новом приложение)
Fn + Alt + F4 = Завершения работы, перезагрузка
Win + C = копировать
Win + V = вставить
Ctrl + Z = отменить последнее действие
Ctrl + Shift + Z = возвращать отмененную действию
Ctrl + = увелечение машстаба
Ctrl - = уменьшение масштаба
Ctrl + A = выделить всё
Shift + Alt = Win + Space = меняет язык
Crtl + S = для сохранение
Ctrl + A, then Ctrl + Shift + Alt + L = делает код читабельным
