Flask - микрофреймворк для создания сайтов

Клиент отправляет запрос к серверу, в это же время действует DNS - сервер, присваивает ip

Для того что бы фреймворк взаимодействовал с сервером используется, wsgi

WSGI - Web Server Gateway Interface - взаимодействует между Python программой, выполняющейся на стороне сервера, и самим веб-сервером

Все фреймворки работают многопоточна, как так к серверу приходит много запроса и фреймворк должен всех запросов обрабатывать


pip install Flask - чтобы установить пакет
